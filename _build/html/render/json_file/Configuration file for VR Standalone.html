VR Standalone Profile document¶

> analyze ：We analyze the information needed in the scene and save it in task.json for further analysis and processing

### 1.task.json

> Store scene analysis results, rendering settings and other information

task.json

```json {

“software_config”: {

“plugins”: {}, “cg_version”: “standalone_vray5.00.05”, “cg_name”: “VR Standalone”

}, “scene_info”: {}, “task_info”: {

“multi_node”: “0”, “tiles”: “1”, “edit_name”: “”, “project_id”: “0”, “job_stop_time”: “86400”, “stop_after_test”: “1”, “concurrent_tasks”: “1”, “frames_per_task”: “1”, “project_name”: “”, “time_out”: “43200”, “cg_id”: “2008”, “input_cg_file”: “D:/houdini/cg_file/test2014vr_vraystandaloneaCopy.vrscene”,

“is_distribute_render”: “1”,

“distribute_render_node”: “3”

}

}¶
