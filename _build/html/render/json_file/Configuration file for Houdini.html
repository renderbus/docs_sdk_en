Houdini Configuration¶

> Analyze houdini scene and save the output as task.json, asset.json,upload.json and tips.json.

### 1.task.json

> File to save the analysis result of the scene, including scene name, cg software version, render settings, etc.

task.json

```json {

“scene_info_render”: {
“rop_node”: [
{

“node”: “/out/mantra1”, “frames”: “1-10[1]”, “option”: “-1”, “render”: “1”, “height”: “720”,

“width”: “1280”

}

], “geo_node”: [], “distributedsim_node”: [

{

“node”: “/out/distributedsim”, “output_driver”: “/obj/distribute_flattank/save_slices”, “render”: “0”, “simControlName”: “/obj/flattank_sim/DISTRIBUTE_flattank_CONTROLS”, “output_file”: “/geo/flip_test_slice4.flattank.0.1.bgeo.sc”, “num_slices”: “4”, “option”: “1”, “frames”: “1-240[1]”, “sliceType”: “particle”
